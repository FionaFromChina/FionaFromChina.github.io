###事件处理

  1. 过程：捕获，处理，冒泡
  2. IE8直接事件冒泡，且只冒泡到document对象；IE9+ 支持DOM2级事件处理且会冒泡到window对象；
  3. 捕获：window->具体，冒泡：具体->window；
  4. 虽然DOM02级别规定了是3个过程，但是一般执行的时候会在捕获或者冒泡的过程中顺便执行事件，所以DOM2级的第3个参数，一个bool值，
    • true：在捕获阶段调用事件处理； window->具体
    • false:在冒泡阶段调用事件处理；具体->window
      一般情况下这个bool值默认是false,都是冒泡的时候再执行事件。
