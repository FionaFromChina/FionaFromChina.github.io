20170722-Mac系统打不开软件

Jul 22, 2017 | Hits

打不开XXX，因为该文件已损坏，请移到废纸篓。

这种提示不是因为下载的资源有问题，而是因为下载的地方不被Mac系统认可，所以才会弹出提示。

想要打开这种文件，需要在【设置】-【安全性与隐私】里面【允许从以下位置下载的引用】，支持任何来源才可以。

但最新的系统升级后，没有了这个选项，需要关闭设置，在命令行输入命令手动打开。

注意，如果打开后，觉得不安全，又恢复了之前前两个安全设置，那么【任何来源】这个设置也会消失，下次想要打开其他来源的软件，需要再在命令行手动运行打开。
